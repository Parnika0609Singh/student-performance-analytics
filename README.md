# Student Performance Analytics Dashboard

## Overview
This project analyzes student academic performance using real-world data to identify trends and at-risk students.

## Features
- Data cleaning and preprocessing
- KPI engineering (attendance, study hours, marks)
- Performance score calculation
- At-risk student detection using statistical thresholding
- Correlation analysis
- Data visualization

## Dataset
Student Performance Dataset (UCI / Kaggle)

## Key Insights
- ~14% students identified as at-risk
- Study hours and marks show strong correlation
- Attendance moderately impacts performance

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
